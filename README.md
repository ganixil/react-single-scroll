# React Router Single Page Scroll App

This project is a simple React single-page application that uses **react-router-dom** for routing while maintaining a **scroll-based layout** (Home, About, Contact sections on one page).

## Install dependencies
```bash
npm install react-router-dom
```



---

# Running the App

Start the development server:

```bash
npm start
```

Then open:

```text
http://localhost:3000
```


---

# Build for Production

```bash
npm run build
```

Output will be in:

```text
/build
```


---

# 📌 Notes

- This is a hybrid approach (Router + scroll page)
- Router is mainly used for URL tracking and deep linking
- Sections are still rendered on a single page
